### Data Types of Python

##### Variable(변수)

    a_string = "like this" (문자열은 ''나 ""로 감싸줘야한다.)
    a_number = 3
    a_boolean = True
    a_float = 3.12
    a_none = None (존재하지 않는다 라는 뜻 = 없다)

    print(type(a_number)) = int
    print(type(a_float)) = float
    print(type(a_string)) = str
    print(type(a_boolean)) = bool
